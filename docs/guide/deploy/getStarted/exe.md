# Linux 可执行文件部署

1. 点击下载一键包 <download version="linux" />，仅支持 Debain 10、Debain 12、Ubuntu20.04、Ubuntu22.04 及以上系统。
2. 解压，运行 <download version="linux" showExeName />，成功运行后可以进入[下一节](/guide/deploy/console/)。

# Windows 可执行文件部署

1. 点击下载一键包 <download version="win32" />，仅支持 Windows 10、Windows Server 2016 及以上系统。
2. 解压，运行 <download version="win32" showExeName />，如下图成功运行后可以进入[下一节](/guide/deploy/console/)。
